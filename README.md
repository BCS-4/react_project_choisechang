## 사이트 소개

나만의 PFP 생성 및 거래 할 수 있는 사이트입니다.

## 특징 및 기능

1. Home : 지금까지 생성된 총 PFP를 보여준다.

2. My NFTs : 내가 보유하고 있는 PFP를 보여준다. PFP 생성 버튼 및 판매 권한 부여 버튼이 있다. 금액을 입력한 후 등록 버튼을 누르면 입력한 금액으로 판매 상태가 된다. 판매 상태로 등록된 PFP는 가격이 표시 된다. PFP를 클릭하면 상세정보를 볼 수 있다.

3. Sale NFTs : 판매 상태인 모든 PFP를 보여준다. 표시된 가격 버튼을 누르면 구매 할 수 있다. PFP를 클릭하면 상세정보를 볼 수 있다.
